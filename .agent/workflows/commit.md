---
description: Gợi ý commit message chuẩn và thực hiện commit
---

Tôi sẽ giúp bạn tạo một commit message chuyên nghiệp theo chuẩn Conventional Commits và thực hiện commit các thay đổi hiện tại.

### Các bước thực hiện:

1. **Kiểm tra thay đổi:**
   // turbo
   `git status`

2. **Phân tích thay đổi:**
   Tôi sẽ xem xét các file đã thay đổi để hiểu nội dung.

3. **Gợi ý Message:**
   Tôi sẽ đề xuất 3 options message (feat, fix, refactor...) và chờ bạn xác nhận.

4. **Thực hiện Commit:**
   Sau khi bạn chọn, tôi sẽ chạy:
   `git commit -m "[message]"`
