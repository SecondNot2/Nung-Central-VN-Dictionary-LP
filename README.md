# NungDic - Tay Nung Language Dictionary (Landing Page)

Đây là dự án **Landing Page** chính thức cho [**NungDic - Từ điển ngôn ngữ Tày & Nùng**](https://nungdic-app.vercel.app). Một ứng dụng hiện đại giúp bảo tồn và phổ biến ngôn ngữ dân tộc Tày và Nùng.

## 🚀 Tính năng chính

- **Direct App Redirection**: Liên kết trực tiếp tới [ứng dụng NungDic thực tế](https://nungdic-app.vercel.app//).
- **AI Engine Showcase**: Giới thiệu công nghệ dịch thuật AI mạnh mẽ (MegaLLM/Gemini) với giao diện tương tác.
- **Stats Dashboard**: Hiển thị số liệu thực tế về quy mô kho từ vựng và cộng đồng.
- **Cộng đồng & Thảo luận**: Giới thiệu hệ thống thảo luận đa tầng và đóng góp từ vựng.
- **Văn hóa & Từ vựng**: Tích hợp câu chuyện và từ vựng nổi bật hàng ngày.
- **Giao diện Neo-Brutalism**: Thiết kế hiện đại, táo bạo với bảng màu Earth & Bamboo.
- **Responsive & Premium**: Tối ưu hóa trải nghiệm trên mọi thiết bị với các hiệu ứng micro-animations.

## 🛠 Công nghệ sử dụng

- **React 19** + **TypeScript**
- **Vite** (Build tool cực nhanh)
- **Lucide React** (Icons)
- **Vanilla CSS** (Custom styling)

## 📦 Cài đặt và Chạy thử

### Yêu cầu hệ thống

- Node.js (phiên bản 18 trở lên)
- npm hoặc yarn

### Các bước thực hiện

1. Clone dự án:

   ```bash
   git clone https://github.com/SecondNot2/Nung-Central-VN-Dictionary-LP.git
   cd Nung-Central-VN-Dictionary-LP
   ```

2. Cài đặt dependency:

   ```bash
   npm install
   ```

3. Chạy môi trường phát triển:

   ```bash
   npm run dev
   ```

4. Biểu diễn sản phẩm (Production build):
   ```bash
   npm run build
   npm run preview
   ```
